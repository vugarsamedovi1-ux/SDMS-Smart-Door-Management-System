# 🔐 SDMS - ჭკვიანი კარის მართვის სისტემა

## 📋 პროექტის შესახებ

**SDMS (Smart Door Management System)** არის IoT-ზე დაფუძნებული ჭკვიანი კარის მართვის სისტემა, რომელიც შექმნილია საცხოვრებელი და სამუშაო სივრცეების უსაფრთხოების გასაუმჯობესებლად. სისტემა იყენებს **ESP32** მიკროკონტროლერს, **MG995R Servo Motor-ს**, **Reed Sensor-ს**, **Push Button-ს**, **LED ინდიკატორებს**, **Buzzer-ს** და **Blynk IoT Cloud** პლატფორმას, რაც უზრუნველყოფს კარის დისტანციურ მართვას, რეალურ დროში მონიტორინგს და უსაფრთხოების დარღვევის შემთხვევაში ავტომატურ შეტყობინებებს.

---

## 🎓 აკადემიური ინფორმაცია

- **უნივერსიტეტი:** ილიას სახელმწიფო უნივერსიტეტი
- **ფაკულტეტი:** ბიზნესის, ტექნოლოგიებისა და განათლების ფაკულტეტი
- **ტექნოლოგიური სკოლა:** კომპიუტერული ინჟინერია
- **საგანი:** CE490B – დამამთავრებელი საინჟინრო პროექტი
- **პროექტის ტიპი:** Final Engineering Project
- **წელი:** 2026
