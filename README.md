

# ­Ъћљ SDMS - рЃГрЃЎрЃЋрЃўрЃљрЃюрЃў рЃЎрЃљрЃарЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ

## ­ЪЊІ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ

**SDMS (Smart Door Management System)** рЃљрЃарЃўрЃА IoT-рЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃГрЃЎрЃЋрЃўрЃљрЃюрЃў рЃЎрЃљрЃарЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ, рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃў рЃАрЃљрЃфрЃ«рЃЮрЃЋрЃарЃћрЃЉрЃћрЃџ рЃЊрЃљ рЃАрЃљрЃЏрЃБрЃерЃљрЃЮ рЃАрЃўрЃЋрЃарЃфрЃћрЃћрЃЉрЃерЃў рЃЎрЃљрЃарЃўрЃА рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃўрЃА рЃњрЃљрЃАрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃћрЃЉрЃџрЃљрЃЊ. рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃерЃў рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ ESP32 рЃЏрЃўрЃЎрЃарЃЮрЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃћрЃарЃў, рЃАрЃћрЃарЃЋрЃЮ рЃФрЃарЃљрЃЋрЃљ, Reed рЃАрЃћрЃюрЃАрЃЮрЃарЃў рЃЊрЃљ Blynk рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃарЃў рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ, рЃарЃљрЃф рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃЎрЃљрЃарЃўрЃА рЃарЃћрЃљрЃџрЃБрЃа рЃЊрЃарЃЮрЃерЃў рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃА рЃЊрЃљ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃўрЃА рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃА.

### ­ЪјЊ рЃљрЃЎрЃљрЃЊрЃћрЃЏрЃўрЃБрЃарЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ

* **рЃБрЃюрЃўрЃЋрЃћрЃарЃАрЃўрЃбрЃћрЃбрЃў:** рЃўрЃџрЃўрЃљрЃАрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃЏрЃгрЃўрЃцрЃЮ рЃБрЃюрЃўрЃЋрЃћрЃарЃАрЃўрЃбрЃћрЃбрЃў
* **рЃцрЃљрЃЎрЃБрЃџрЃбрЃћрЃбрЃў:** рЃЉрЃўрЃќрЃюрЃћрЃАрЃўрЃА, рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃњрЃљрЃюрЃљрЃЌрЃџрЃћрЃЉрЃўрЃА рЃцрЃљрЃЎрЃБрЃџрЃбрЃћрЃбрЃў
* **рЃАрЃћрЃЦрЃфрЃўрЃљ:** рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃБрЃџрЃў рЃўрЃюрЃЪрЃўрЃюрЃћрЃарЃўрЃљ
* **рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃбрЃўрЃърЃў:** рЃЊрЃљрЃЏрЃљрЃЏрЃЌрЃљрЃЋрЃарЃћрЃЉрЃћрЃџрЃў рЃЎрЃБрЃарЃАрЃўрЃА рЃЊрЃўрЃќрЃљрЃўрЃюрЃўрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃў A
* **рЃгрЃћрЃџрЃў:** 2026
* **рЃЦрЃљрЃџрЃљрЃЦрЃў:** рЃЌрЃЉрЃўрЃџрЃўрЃАрЃў, рЃАрЃљрЃЦрЃљрЃарЃЌрЃЋрЃћрЃџрЃЮ

### ­ЪЉЦ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃњрЃБрЃюрЃЊрЃў

* **рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃў:**

  * рЃЋрЃБрЃњрЃљрЃа рЃАрЃљрЃЏрЃћрЃЊрЃЮрЃЋрЃў (рЃљрЃърЃљрЃарЃљрЃбрЃБрЃарЃљ & рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ)
  * рЃћрЃџрЃ»рЃБрЃю рЃ«рЃљрЃАрЃўрЃћрЃЋрЃў (рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџрЃў рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃљ & рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃЋрЃўрЃЌрЃљрЃарЃћрЃЉрЃљ)
* **рЃЎрЃЮрЃюрЃАрЃБрЃџрЃбрЃљрЃюрЃбрЃћрЃЉрЃў:**

  * рЃЊрЃљрЃЋрЃўрЃЌ рЃЕрЃ«рЃљрЃўрЃФрЃћ
  * рЃњрЃўрЃЮрЃарЃњрЃў рЃЏрЃЮрЃЊрЃћрЃЉрЃљрЃФрЃћ

## ­ЪїЪ рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў

### рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў

* РюЁ **рЃарЃћрЃљрЃџрЃБрЃа рЃЊрЃарЃЮрЃерЃў рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃў:** Reed рЃАрЃћрЃюрЃАрЃЮрЃарЃў рЃЎрЃљрЃарЃўрЃА рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃўрЃА рЃљрЃюрЃљрЃџрЃЮрЃњрЃўрЃБрЃарЃў рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃў
* ­ЪЊ▒ **рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃарЃў рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃў:** Blynk рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ рЃЊрЃўрЃАрЃбрЃљрЃюрЃфрЃўрЃБрЃарЃў рЃгрЃЋрЃЊрЃЮрЃЏрЃљ
* ­Ъћћ **рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃўрЃА рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃў:** рЃФрЃљрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ рЃњрЃљрЃ«рЃАрЃюрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃ«рЃЏрЃЮрЃЋрЃљрЃюрЃў рЃљрЃџрЃљрЃарЃЏрЃў рЃЊрЃљ рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃљ
* ­Ъћњ **рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃЕрЃљрЃЎрЃћрЃбрЃЋрЃљ:** Servo Motor-рЃўрЃЌ рЃАрЃарЃБрЃџрЃў рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃАрЃљрЃЎрЃћрЃбрЃўрЃА рЃЏрЃћрЃЦрЃљрЃюрЃўрЃќрЃЏрЃў
* ­ЪњА **LED рЃўрЃюрЃЊрЃўрЃЎрЃљрЃбрЃЮрЃарЃў:** рЃЏрЃгрЃЋрЃљрЃюрЃћ/рЃгрЃўрЃЌрЃћрЃџрЃў LED-рЃћрЃЉрЃўрЃЌ рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃў рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃў
* ­Ъїљ **Wi-Fi рЃЎрЃљрЃЋрЃерЃўрЃарЃў:** IoT рЃдрЃарЃБрЃЉрЃџрЃўрЃА рЃАрЃћрЃарЃЋрЃўрЃАрЃћрЃЉрЃЌрЃљрЃю рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ

### рЃбрЃћрЃЦрЃюрЃўрЃЎрЃБрЃарЃў рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў

* **рЃЏрЃўрЃЎрЃарЃЮрЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃћрЃарЃў:** ESP32 (Dual-core 240MHz)
* **рЃЎрЃЮрЃЏрЃБрЃюрЃўрЃЎрЃљрЃфрЃўрЃљ:** Wi-Fi 802.11 b/g/n, Bluetooth
* **рЃФрЃљрЃџрЃљ:** 5V DC
* **рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃў:** Servo Motor (MG995R)
* **рЃАрЃћрЃюрЃАрЃЮрЃарЃћрЃЉрЃў:** Reed рЃЏрЃљрЃњрЃюрЃўрЃбрЃБрЃарЃў рЃАрЃћрЃюрЃАрЃЮрЃарЃў
* **рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃљ:** Buzzer + LED рЃўрЃюрЃЊрЃўрЃЎрЃљрЃбрЃЮрЃарЃћрЃЉрЃў

## ­ЪЈЌ№ИЈ рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљ

### рЃцрЃћрЃюрЃЮрЃЋрЃљрЃюрЃў рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљ

1. **рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃўрЃА рЃцрЃћрЃюрЃљ**

   * Blynk рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃарЃў рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ
   * рЃцрЃўрЃќрЃўрЃЎрЃБрЃарЃў рЃдрЃўрЃџрЃљрЃЎрЃў (рЃЏрЃљрЃюрЃБрЃљрЃџрЃБрЃарЃў рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃў)

2. **рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃўрЃА рЃЉрЃўрЃарЃЌрЃЋрЃў**

   * ESP32 Wi-Fi рЃЏрЃўрЃЎрЃарЃЮрЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃћрЃарЃў

3. **рЃњрЃљрЃЊрЃљрЃбрЃљрЃюрЃўрЃА рЃЏрЃћрЃЦрЃљрЃюрЃўрЃќрЃЏрЃћрЃЉрЃў**

   * Servo Motor рЃАрЃљрЃЎрЃћрЃбрЃў
   * LED рЃўрЃюрЃЊрЃўрЃЎрЃљрЃбрЃЮрЃарЃћрЃЉрЃў
   * Buzzer рЃљрЃџрЃљрЃарЃЏрЃў

4. **рЃАрЃћрЃюрЃАрЃЮрЃарЃБрЃџрЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ**

   * Reed рЃАрЃћрЃюрЃАрЃЮрЃарЃў (рЃЎрЃљрЃарЃўрЃА рЃЏрЃЊрЃњрЃЮрЃЏрЃљрЃарЃћрЃЮрЃЉрЃљ)

5. **рЃЦрЃАрЃћрЃџрЃў рЃЊрЃљ рЃдрЃарЃБрЃЉрЃџрЃўрЃА рЃцрЃћрЃюрЃљ**

   * Wi-Fi рЃЎрЃљрЃЋрЃерЃўрЃарЃў
   * Blynk Cloud Server

## ­ЪЊі рЃЊрЃўрЃљрЃњрЃарЃљрЃЏрЃћрЃЉрЃў

рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃбрЃћрЃЦрЃюрЃўрЃЎрЃБрЃа рЃАрЃЦрЃћрЃЏрЃћрЃЉрЃА:

1. **3D рЃЊрЃўрЃќрЃљрЃўрЃюрЃў** - SDMS рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃўрЃА рЃЏрЃЮрЃЊрЃћрЃџрЃў
2. **рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃўрЃА рЃЊрЃўрЃљрЃњрЃарЃљрЃЏрЃљ** - рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃџрЃљрЃњрЃћрЃЉрЃљ
3. **рЃћрЃџрЃћрЃЦрЃбрЃарЃЮ рЃЊрЃўрЃљрЃњрЃарЃљрЃЏрЃљ** - рЃФрЃљрЃЉрЃЋрЃўрЃА рЃњрЃљрЃюрЃљрЃгрЃўрЃџрЃћрЃЉрЃљ
4. **рЃАрЃЦрЃћрЃЏрЃљрЃбрЃБрЃарЃў рЃЊрЃўрЃљрЃњрЃарЃљрЃЏрЃљ** - рЃЎрЃљрЃЋрЃерЃўрЃарЃўрЃА рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃў
5. **PCB рЃЊрЃўрЃљрЃњрЃарЃљрЃЏрЃљ** - рЃЉрЃћрЃГрЃЊрЃБрЃарЃў рЃгрЃарЃўрЃА рЃЊрЃўрЃќрЃљрЃўрЃюрЃў
6. **рЃцрЃБрЃюрЃЦрЃфрЃўрЃБрЃарЃў рЃЊрЃўрЃљрЃњрЃарЃљрЃЏрЃљ** - рЃАрЃўрЃАрЃбрЃћрЃЏрЃўрЃА рЃцрЃџрЃЮрЃБ


## ­Ъћњ рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃљ

### рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃўрЃА рЃцрЃћрЃюрЃћрЃЉрЃў

1. **рЃцрЃўрЃќрЃўрЃЎрЃБрЃарЃў рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃљ**

   * рЃЏрЃћрЃЦрЃљрЃюрЃўрЃЎрЃБрЃарЃў рЃАрЃљрЃЎрЃћрЃбрЃўрЃА рЃЉрЃџрЃЮрЃЎрЃўрЃарЃћрЃЉрЃљ
   * рЃФрЃљрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ рЃњрЃљрЃ«рЃАрЃюрЃўрЃА рЃЊрЃћрЃбрЃћрЃЦрЃфрЃўрЃљ

2. **рЃџрЃЮрЃњрЃўрЃЎрЃБрЃарЃў рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃћрЃЉрЃљ**

   * рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃўрЃА рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљ

   * рЃ«рЃЏрЃЮрЃЋрЃљрЃюрЃў рЃљрЃџрЃљрЃарЃЏрЃў (Buzzer)
